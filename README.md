# 百度网盘图片批量下载工具

按图片编号或文件名在百度网盘里搜索图片，并批量下载到本地。适合不会命令行的新手使用。

## 最简单的用法

1. 双击 `点我启动.bat`。
2. 浏览器会自动打开本地页面。
3. 点击“浏览器登录”，在 Edge 或 Chrome 里登录百度网盘。
4. 粘贴图片编号或文件名。
5. 先点“仅查询不下载”，确认能找到后再点“开始下载”。

下载后的图片默认保存在工具目录下的 `downloads` 文件夹。

## 首次使用需要什么

- Windows 电脑
- Python 3.8 或更高版本
- Edge 或 Chrome 浏览器
- 已能正常登录的百度网盘账号

`点我启动.bat` 会自动检查并安装依赖。第一次启动时请保持网络连接。

## 页面里的几个设置

- `网盘搜索目录`：百度网盘里的路径。默认 `/` 表示全盘搜索。
- `保存到电脑的文件夹`：默认 `downloads`。相对路径会保存在本工具目录下。
- `同时搜索子文件夹`：默认开启，适合不知道图片在哪个子目录的情况。

## 编号怎么填

支持下面几种写法：

```text
SFD202603072351
SFD202603054091
```

也可以直接粘贴一整串：

```text
SFD202603072351,SFD202603054091 SFD202603012222
```

只填编号也可以，工具会自动匹配同名的图片文件，例如 `SFD202603072351.jpg`、`SFD202603072351.png`。

## 常见问题

**提示未登录或 Cookie 过期怎么办？**  
重新点击页面里的“浏览器登录”，在 Edge 或 Chrome 里登录百度网盘。

**找不到图片怎么办？**  
先确认“网盘搜索目录”填的是百度网盘里的目录，例如 `/` 或 `/商品图片`，不是电脑上的 `D:\图片`。

**下载很慢怎么办？**  
百度网盘对非会员下载速度可能有限制，这是网盘侧限制。可以稍后重试。

**自动登录失败怎么办？**  
展开页面里的“自动登录失败时手动粘贴 Cookie”，按提示粘贴包含 `BDUSS=` 的完整 Cookie。

## 命令行用法

新手建议使用网页。熟悉命令行时也可以直接运行：

```bash
python main.py download
python main.py download --dry-run
python main.py download 图片1.jpg 图片2.png
```

配置保存在 `config.json`，编号列表保存在 `names.txt`。`config.json` 包含登录信息，请不要发给别人。
